# FWA-Getter CLI
Connect to the [The FWA](https://thefwa.com) and shows the information of an specific agency right into your console.

### Installation
```bash
npm i -g fwagetter
```

### Usage
Only need to define the name of the agency to get the information:
```sh
fwagetter mediamonks
```
